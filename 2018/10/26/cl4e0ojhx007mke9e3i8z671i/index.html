使用 StoryBook 搭建 UI 组件库

StoryBook 为UI组件开发提供开发环境，你可有实现不同状态下的UI可视化，并且可以实现交互。

实现 Vue UI 组件库

初始化项目

vue init webpack my-vue-project. // 通过 vue cli 初始化项目

cd my-vue-project

npm i -g @storybook/cli // 全局安装 storybook

getstorybook // 生成

npm run storybook //运行
